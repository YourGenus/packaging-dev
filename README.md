# Genus Packaging Development  
Sustainable packaging combining Aluminum Shells with PCR Inserts, designed across identity-driven Genres: **Citizen, Athlete, Dreamer, Metropolitan, and Traveler**. This repository explores early prototypes, color systems, ergonomic studies, and AI‑assisted design workflows.

## Purpose
Genus develops next‑generation packaging using circular materials and modern design tools. This repo supports:
- Aluminum + PCR hybrid structures  
- 2D and 3D prototyping  
- Color and Genre identity systems  
- AI‑assisted concept exploration  
- Early product foundations for a future DTC platform
